TicTacToe using Artificial Intelligence
=======================================
A game of TicTacToe implemented using Min-Max algorithm.

### Compile
```
- Visual Studio 2015
```

### Run
```
- ./tictactoe
```

### Usage

1. 모드 선택(사람vs사람 또는 사람vs컴퓨터)
  - 사람 vs 컴퓨터 선택 시 랭킹 시스템 이용가능
  - 랭킹 시스템이란 컴퓨터랑 경기 했을 경우 게임 시간 측정 가능한 기능<br>
